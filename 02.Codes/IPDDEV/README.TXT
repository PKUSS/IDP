This project is created by a student from pku, to make some experiments and developing practice.
With the purpose to develope a idp system, the project is based on hibernate, 
spring and struts2 frameworks.   
修改'demo/src/main/resources/spring/data-access.properties 修改用户名密码即可
编译命令：mvn clean package
在target目录下即可找到demo.war

Using java ee6,JPA, etc.
